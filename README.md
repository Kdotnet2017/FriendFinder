# Friend Finder App
 
 A compatibility-based "FriendFinder" application that displays the best overall match. 

### How the app works: 
* nodejs - express 
* Folder structure  
    - app
      - data
        - friends.js
      - public
        - home.html
        - survey.html
      - routing
        - apiRoutes.js
        - htmlRoutes.js
    - node_modules
    - package.json
    - server.js
